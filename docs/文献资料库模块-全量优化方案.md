当前模块已经完成了「文献条目展示 + 手动添加 + 下载」的基础闭环，但距离课题组真正能用的文献管理工具还有差距：录入效率低、筛选维度少、科研属性弱、和其他模块割裂。下面从列表管理端、录入发布端、详情阅读端、全系统联动四个维度做完整优化，最终实现「快速入库→高效检索→深度阅读→对比分析→成果复用」的完整科研文献流。

二、添加弹窗优化：从「手动填表」升级为「多方式智能入库」
手动录入是效率最低的方式，科研场景下 90% 的文献都是通过导入入库。弹窗改为Tab 多模式添加，覆盖所有常用录入场景，大幅降低录入成本。
弹窗顶部增加 4 个 Tab 切换
Tab1：手动添加（保留现有能力，字段补全）
在现有字段基础上，补全科研必需元数据，分为「基础信息」「附件信息」「其他设置」三个区块：
基础信息
文献标题 *
作者（支持多人，逗号分隔）
期刊 / 会议名称
发表年份
DOI 号
引用量
文献类型（下拉选择：期刊 / 会议 / 学位论文 / 综述 / 报告）
内容信息
摘要（多行文本框）
关键词 / 标签（下拉选择 + 自定义新增，不再纯手动输入逗号分隔）
原文链接
附件与设置
上传 PDF 原文（支持拖拽上传）
是否共享全组（默认开启）
备注
Tab2：DOI 快速导入（最高频，首选）
输入框：粘贴文献 DOI 号
「自动提取」按钮：调用学术元数据接口，自动拉取标题、作者、期刊、年份、摘要、引用量
提取成功后自动填充到表单，用户确认后一键保存
价值：不用手动填任何信息，10 秒录入一篇文献，准确率 100%
Tab3：BibTeX 批量导入（适配文献管理工具）
两种方式：粘贴 BibTeX 文本 / 上传 .bib 文件
自动解析所有文献条目，预览列表后可勾选批量入库
自动去重：根据 DOI 判断重复，重复文献提示跳过或覆盖
价值：直接从 Zotero、EndNote、Mendeley 导出导入，整组文献库一键迁移
Tab4：PDF 智能解析入库（联动文档智能解析）
上传论文 PDF 文件
调用 AI 自动提取：标题、作者、摘要、关键词、发表年份、核心指标
提取后生成预览，用户修正后一键入库
价值：扫描版、本地 PDF 文献快速归档，不用手动敲信息
录入体验细节优化
标签智能联想：输入标签时自动联想已有标签，保持分类统一，避免出现「城市安全」「城市安防」这类同义不同名的标签
作者自动补全：输入作者时联想团队成员、领域知名学者，提升录入速度
自动去重校验：提交时根据 DOI / 标题查重，重复文献给出提示，避免库内冗余
上传者自动填充：默认填充当前登录用户，无需手动选择
三、新增：文献详情页（阅读 + 笔记 + 引用一体化）
点击「查看」进入详情页，替代简单的弹窗预览，做成完整的文献阅读工作台：
左侧元数据区：完整展示所有文献信息（标题、作者、期刊、年份、DOI、引用量、标签、摘要）
右侧 PDF 预览区：在线预览 PDF 原文，支持翻页、缩放、目录导航
底部功能区：
笔记区：可记录阅读笔记，支持 Markdown，笔记自动保存
引用格式：一键复制 GB/T 7714、APA、IEEE 三种格式，写论文直接粘贴
相关文献推荐：基于标签、作者推荐同领域文献
操作：下载 PDF、加入对比、收藏、分享链接
阅读状态：标记「未读 / 在读 / 已读」，记录阅读进度
四、全系统联动：融入整体科研工作流
这是和普通文件管理的核心区别，让文献库成为整个系统的科研数据底座。
1. 联动「文献对比分析」
共用同一张文献数据表，数据同源，无需重复录入
列表选中文献后一键跳转对比，自动带入选中条目
对比分析页可一键将文献加入收藏、归档到分组
2. 联动「文档智能解析」
上传 PDF 自动调用解析能力提取元数据，实现「上传即入库」
文献详情页可直接跳转智能解析，生成文献总结、核心观点提炼
3. 联动「团队工作周报」
详情页增加「插入本周周报」按钮，自动将文献信息 + 阅读笔记追加到学生当周周报的文献调研板块
不用学生再复制粘贴，周报撰写效率大幅提升
4. 联动「项目管理」
文献可关联到具体科研项目（如工地安全检测项目）
项目档案中自动汇总所有相关参考文献，结题时一键导出参考文献清单
5. 联动「团队共享文件库」
文献 PDF 可一键保存到共享文件库对应分类
共享文件库中的论文 PDF，可一键导入到文献资料库
6. 权限联动
复用现有 RBAC 体系：
导师 / 管理员：全库管理权限，可删除、编辑所有文献
学生：可管理自己上传的文献，可查看、下载全组共享文献
访客：仅可查看文献元数据，不可下载 PDF
