（列表页 + 上传弹窗双端升级，适配 AI 训练大文件场景，打通数据标注、模型训练、项目成果全链路）

- 上传限制读取 `.env` / 网关配置

### 使用说明
1. 用 `python start_web.py` 启动（不要直接双击 html），大文件分片与 ZIP 样例直链才可用
2. 浏览器 **Ctrl+F5** 强制刷新
3. 生产环境可在 `.env` 配置 `DATASET_UPLOAD_TOKEN`（或复用 `ANNOTATION_UPLOAD_TOKEN`）

### API 一览
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/dataset/health` | 能力探测 |
| POST | `/api/dataset/init` | 初始化/秒传 |
| POST | `/api/dataset/chunk` | 上传分片 |
| POST | `/api/dataset/complete` | 合并校验 |
| GET | `/api/dataset/status` | 断点查询 |
| GET | `/api/dataset/inspect` | 服务端解析 |
| GET | `/api/dataset/sample` | ZIP 样例图 |
| GET | `/api/dataset/download` | 整文件下载 |
