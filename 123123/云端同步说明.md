# 全站数据同步（已开通）

现在**不需要再执行 SQL**。系统会自动把共享数据同步到云端（复用现有 Supabase `patents` 表中的同步占位记录）。

## 怎么用

1. 强制刷新页面：`Ctrl + F5`
2. 登录后右下角会出现「已同步云端数据 … 项」
3. 任意人增删改（成员、任务、项目等），其他人约 15 秒内或刷新后即可看到

## 会同步什么

- 团队成员档案、账号权限、项目、任务、周报
- 通知/新闻/会议、文献/数据集/报告、共享文件列表
- 标准/软著、系统配置、操作日志等

## 不会同步什么

- API Key、登录会话
- 成员头像大图（base64 过大，同步时会清空头像字段，姓名等资料仍同步）
- 共享文件真实二进制（只同步文件列表；文件本体仍在各浏览器本地）

## 专利台账

同步占位记录不会出现在专利列表里（已自动过滤）。
