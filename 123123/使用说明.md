# Excel 智能处理模块 - 使用说明

## 🎯 功能概述

Excel 智能处理模块已重构完成，支持以下功能：

1. **文件上传**：支持拖拽上传和点击上传，可多文件同时上传
2. **自然语言输入**：通过文本框输入数据处理需求
3. **自动完成**：输入时显示建议命令
4. **快速指令**：一键插入常用分析命令
5. **AI 处理**：调用通义千问 API 处理数据并生成图表
6. **图表展示**：使用 ECharts 展示处理结果

## 🚀 使用方式

### 方式 1：演示模式（无需配置）

直接打开 `index.html` 文件即可使用演示模式：

1. 上传 Excel 文件
2. 输入数据处理需求
3. 点击"🚀 发送"或"🤖 AI 智能处理"
4. 系统会生成演示图表（随机数据）

**注意**：演示模式使用随机数据，不会真正调用 AI API。

### 方式 2：完整功能（需要 Node.js 代理）

要真正调用通义千问 API，需要运行代理服务器：

#### 步骤 1：安装 Node.js

如果未安装 Node.js，请访问 https://nodejs.org 下载并安装。

#### 步骤 2：启动代理服务器

打开命令提示符（CMD）或 PowerShell，进入项目目录：

```bash
cd C:\Users\Admin\Desktop\123123
```

运行代理服务器：

```bash
node server.js
```

看到以下输出表示服务器启动成功：

```
🚀 代理服务器运行在 http://localhost:3000
📡 代理端点：http://localhost:3000/api/aliyun
⚠️  按 Ctrl+C 停止服务器
```

#### 步骤 3：使用完整功能

1. 保持代理服务器运行
2. 打开 `index.html` 文件
3. 上传 Excel 文件
4. 输入数据处理需求
5. 点击发送
6. 系统会调用通义千问 API 并返回真实的分析结果

## ⚠️ 常见问题

### Q1: 为什么显示"API 调用失败"？

**A**: 这是因为浏览器跨域限制。有两种解决方案：

1. **推荐**：运行 Node.js 代理服务器（见方式 2）
2. 使用演示模式（会生成随机数据图表）

### Q2: 代理服务器无法启动？

**A**: 检查以下几点：

1. 确保已安装 Node.js
2. 确保端口 3000 未被占用
3. 检查防火墙设置

### Q3: 如何修改 API Key？

**A**: 在 `index.html` 文件中搜索 `sk-997a88dd8f274103a2e3d427bf29a873`，替换为您自己的 API Key。

或在界面设置中配置（如果支持）。

### Q4: 支持哪些 Excel 格式？

**A**: 支持 `.xlsx` 和 `.xls` 格式。

### Q5: 可以上传多个文件吗？

**A**: 是的，支持多文件同时上传。

## 🎨 界面特性

- **渐变色上传区域**：美观的紫色渐变背景
- **拖拽上传**：支持拖拽文件到上传区域
- **自动完成**：输入时显示建议命令
- **快速指令**：4 个预设快捷指令按钮

- **响应式图表**：自动适应窗口大小

- 柱状图（bar）
- 折线图（line）
- 饼图（pie）
- 散点图（scatter）
- 面积图（area）
- 等等...

## 🔧 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **图表库**: ECharts 5.4.3
- **Excel 处理**: SheetJS (xlsx)
- **AI API**: 通义千问 Qwen3.6-Plus
- **代理服务器**: Node.js (原生 http 模块)
